# my_little_board
기본적인 CRUD기능을 하는 간단한 게시판 페이지입니다.
회원가입, 로그인 패스워드 암호화(bcrypt), Session 기능을 사용하였습니다.

### 개발 환경
- Nextjs(v14.1.0)
- express(v4.18.2)
- Node(v20.9.0)
- mariaDB(v10.6.7)
